# Code for Gendercaremap.org

a simple website that allows users to add/edit resources on a map and allows reviewing them to see how the resource is performing.

The goal is to help trans people find resources. A lot of communities have large lists or random messages / pins in discord channels, this centralizes the data into a map so it's easier to find and review.

# tech stack

The tech stack is made to be simple and runs on a single server.

server : digital ocean droplet, ubuntu server, nginx, letsencrypt

database : sqlite

backend : nodejs nuxt

frontend : vue, nuxt, openstreetmap and leaflet for map functions

style : bootstrap
